    If you are interested in learning a quick and low-tech assessment tool to use in the classroom, you have come to the right place. This course is organized into three modules intended to cover the basics of using Plickers in the classroom. Module One will cover setup of the tool, including everything that you need to have done prior to classroom use. Module Two will go over how to use Plickers in the classroom. Module Three will address tips and troubleshooting
