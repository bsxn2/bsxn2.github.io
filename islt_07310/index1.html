Plicking in the Classroom

Plickers is a low-tech assessment tool that makes checking for understanding fast and engaging.
This course is organized into three modules intended to provide you with the basics of using Plickers in the classroom. Module One will cover the setup of the tool, including everything that you need to have done prior to using it in the classroom. Module Two will go over how to use Plickers in the classroom. Module Three will address tips and troubleshooting to help streamline incorporating the tool.
